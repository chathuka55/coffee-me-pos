# Coffee Me POS - Point of Sale System

A modern, full-featured Point of Sale (POS) system for cafes and restaurants built with React, TypeScript, and Vite.

## Project Overview

Coffee Me POS is a comprehensive point of sale system designed for cafes and restaurants. It includes features for managing inventory, processing orders, table management, customer tracking, and generating reports.

## Technologies Used

This project is built with:

- **Vite** - Next generation frontend tooling
- **TypeScript** - Type-safe JavaScript
- **React** - UI library
- **shadcn-ui** - High-quality UI components
- **Tailwind CSS** - Utility-first CSS framework
- **React Query** - Data fetching and state management
- **React Router** - Client-side routing

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn
- Git

### Installation

Follow these steps to set up the project locally:

```sh
# Step 1: Clone the repository
git clone https://github.com/chathuka55/coffee-me-pos

# Step 2: Navigate to the project directory
cd coffe_me_pos

# Step 3: Install the necessary dependencies
npm install

# Step 4: Start the development server
npm run dev
```

The application will be available at `http://localhost:8080`

### Building for Production

```sh
# Build the project
npm run build

# Preview the production build
npm run preview
```

## Features

- **Point of Sale (POS)** - Process orders with multiple payment methods
- **Inventory Management** - Track and manage product inventory
- **Table Management** - Manage dining tables and reservations
- **Order Management** - View and manage orders
- **Customer Management** - Track customer information
- **Reports & Analytics** - Generate sales reports and analytics
- **Settings** - Configure shop settings and preferences
